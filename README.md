# Класификация на газове чрез данни от IoT сензорна мрежа

Демонстрационен проект по дисциплината **„Анализ на големи масиви данни и откриване на знания“**.

Темата на презентацията е **„Безжични сензорни мрежи за IoT“**, а практическата част показва как данни от сензорна система могат да се използват за обучение на ML модел за класификация.

## Идея на проекта

В реални IoT системи безжични сензорни мрежи могат да събират данни за средата - температура, влажност, газове, замърсяване, движение и други. Тези данни могат да се анализират чрез машинно обучение, за да се откриват състояния, аномалии или рискови ситуации.

Този проект използва данни от химически сензори и обучава модел, който разпознава вида на газа според реакцията на сензорите.

## Dataset

Използван е **Gas Sensor Array Drift Dataset** от UCI Machine Learning Repository.

Dataset-ът съдържа:

- 13 910 измервания;
- 16 химически сензора;
- 128 числови характеристики;
- 6 класа газове:
  - Ethanol
  - Ethylene
  - Ammonia
  - Acetaldehyde
  - Acetone
  - Toluene

Данните се свалят директно от UCI като ZIP архив. `.dat` файловете са във формат, подобен на LIBSVM:

```text
class feature:value feature:value ...
```

Например:

```text
1 1:15596.162100 2:1.868245 ... 128:-2.654529
```

Първото число е класът на газа, а следващите стойности са характеристиките от сензорите.

## Модели

В проекта се обучават два модела:

1. **Random Forest Classifier** - baseline модел от scikit-learn.
2. **FastAI Tabular Learner** - основният модел, обучен чрез FastAI.

## Стартиране на проекта на macOS

### 1. Създаване на virtual environment

```bash
python3 -m venv .venv
```

### 2. Активиране

```bash
source .venv/bin/activate
```

### 3. Инсталиране на зависимости

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### 4. Стартиране във VS Code

Отвори:

```text
notebooks/gas_sensor_classification.ipynb
```

Избери kernel от `.venv` и изпълни клетките отгоре надолу.

## Извод

Проектът демонстрира приложението на машинното обучение при анализ на данни от IoT сензорни мрежи. Подобен подход може да се използва при мониторинг на въздуха, индустриална безопасност, откриване на опасни газове и автоматизирано вземане на решения в IoT системи.
